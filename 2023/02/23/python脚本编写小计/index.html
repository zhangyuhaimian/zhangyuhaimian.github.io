请求方法
http1.0支持

  1. GET
  2. POST
  3. HEAD

http1.1支持

  1. PUT

  3. TRACE
  4. CONNECT

url 协议://主机名.域名/文件夹/文件？参数=值&参数1=值1

Http 请求头

  1. User-Agent 浏览器版本信息
  2. Accept-encoding 浏览器接受的编码
  3. Referer 当前网页是从哪里跳转过来的
  4. Cookie Cookie信息
  5. Location 跳转到哪里
  6. Set-Cookie 设置Cookie信息
  7. WWW-Authenticate 用于身份验证 HTTP Basic等

Http响应状态码

  1. 1xx 信息提示
  2. 2xx 成功
  3. 3xx 重定向
  4. 4xx 客户端错误
  5. 5xx 服务端错误

Python Http请求

  1. GET请求 不带参数 requests.get(url)
    带参数 requests.get(url=url,params=(key1=value1,key2=value2))
    r.url 获取请求url

  2. post请求 不带参数 requests.post(url)
    request.post(url,data={(key1:value1)})

  3. 自定义请求头 headers = {key:value}
    request.get(url,headers=headers)

    修改headers

import requests
url = "https://www.baidu.com"
payload = {'User-Agent':'zhangyuhaimian'}
data = {'wd':'123465'}
r = requests.get(url)
p = requests.get(url,headers=payload)
print(p.url)
print(p.status_code)
print(p.request.headers)

Python-Http响应

  1. 获取响应状态码 r.status_code

  2. 获取响应文本 r.content 二进制数据包含换行 ctrl + / 注释
    r.text 源代码

  3. 获得响应头 r.headers
    请求头 r.request.headers

  4. 输出url r.url

  5. 获取cookie r.cookie

    https verify = False

proxise = {'http': 'http://192.168.1.110:8081'}

r = requests.get(url, proxies=proxise, verify=False)
